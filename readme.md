# Git Course Repository

This repository is created to practice and understand Git and GitHub concepts.

## 📁 Project Structure

- `one/` → Contains basic files for initial practice  
- `two/` → Used for experimenting with Git commands  
- `three/` → Working directory for commits, push, and branch tracking  
- `four/` → Additional testing and practice files  

## 🚀 What I Practiced

- Initializing a Git repository  
- Adding and committing files  
- Understanding staging area  
- Using branches  
- Pushing code to GitHub  
- Tracking remote branches
